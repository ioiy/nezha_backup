# 🛡️ 哪吒面板 V2 自动备份与管理工具 (Nezha Backup)

> 专为 **哪吒探针 (Nezha Panel) V2** 打造的一站式、全自动、带精美交互面板的 S3 数据备份与灾备恢复脚本。即使是 512MB 内存的小鸡也能稳定运行，彻底告别坏档与数据丢失风险！

---

## ✨ 核心亮点 (Features)

- 🖥️ **精美交互式 UI**：告别繁琐的命令行参数，全部操作均可通过纯中文的可视化菜单完成。
- ☁️ **S3 协议全兼容**：基于 Rclone 强力驱动，完美兼容各类云厂商的 S3 存储桶（内置 `no_check_bucket` 机制，支持严格权限的 AK/SK 密钥）。
- 🪶 **512M 低配机福音**：专为小内存机器优化。手动运行时采用极低内存参数；定时运行时自动挂载 `nice` 和 `ionice` 最低优先级，完全不卡面板！
- 🩺 **双重预检机制 (防毁档)**：备份前自动执行 SQLite 数据库健康扫描与可用磁盘空间比对，发现异常立即终止并报警，绝不把损坏或不完整的数据传到云端覆盖。
- 🔒 **AES-256 军事级加密**：内存级流式加密，您的探针数据（包含极度敏感的 GitHub Token、服务器密码）在公有云上也是绝对安全的密文。
- 🗂️ **长效白名单策略 (GFS)**：不仅仅是“保留最近 14 天”，开启 GFS 策略后，脚本会永远保留每个月 1 号的备份文件作为基准存档。
- 🚑 **一键可视化定点恢复**：独创的 S3 备份管理中心！可以直接在面板里列出云端备份，输入序号即可一键全自动下载、解密并覆盖重启面板。
- 🤖 **全能 Telegram 播报**：成功或失败均可推送到 TG，自带详细日志及当前服务器的「剩余磁盘、内存」监控播报。

---

## 🚀 极速安装与使用 (Quick Start)

只需在安装了哪吒面板的服务器上，使用 `root` 用户运行以下一键命令：

```bash
curl -O [https://raw.githubusercontent.com/ioiy/nezha_backup/main/bf.sh](https://raw.githubusercontent.com/ioiy/nezha_backup/main/bf.sh) && chmod +x bf.sh && ./bf.sh
